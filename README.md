<img align="right" alt="Lock Icon" src="data/icons/com.konstantintutsch.Lock.svg">

Process data with GnuPG

_Lock_ is a graphical front-end for GnuPG (GPG) making use of a beautiful LibAdwaita GUI.

Supported are

- Encryption
- Decryption
- Signing
- Verification

of text and files.
